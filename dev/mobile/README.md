# 手机端样品采集

这是手机端第一版静态应用，先用于跑通核心业务。当前默认是开发环境：

```text
应用名：样品采集 Dev
数据库：sample-mobile-db-dev
导出包：sample_sync_dev_时间.zip
```

```text
录入 → 录入详情 → 结束
查看 → 项目列表 → 项目详情 → 具体产品详情 → 结束
同步中心 → 导出/导入 → 结束
```

## 当前能力

- 本地 IndexedDB 保存项目和样品。
- 支持拍照/选择图片。
- 支持录入后进入录入详情。
- 支持按项目查看样品。
- 支持调整已有样品内容。
- 支持导出 zip 数据包。
- 支持导入本应用生成的 zip 数据包。

## 本地预览

可以直接用浏览器打开 `mobile/index.html`。

后续接入 Capacitor 时，再把这个静态应用作为 Web 资源打包成 APK。
