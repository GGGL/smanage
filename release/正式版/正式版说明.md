# 正式版交付目录

这里用于放最终可直接使用的正式版程序。

正式版目标结构：

```text
正式版/
  打开桌面端.exe
  手机端.apk
  数据包/
    手机导出给电脑/
    电脑导出给手机/
  数据备份/
```

使用说明：

- 双击 `打开桌面端.exe` 启动电脑端管理页面。
- 默认使用本机端口 `8765`；如果被占用，会自动改用 `8766` 等后续端口。
- 手机安装 `手机端.apk` 后进行现场采集。
- 手机导出的包放入 `数据包/手机导出给电脑/` 后，在电脑端导入。
- 电脑端导出的包默认保存到 `数据包/电脑导出给手机/`，再传给手机端导入。
- 电脑端导入前自动备份数据库，备份放入 `数据备份/`。

正式版必须使用 production 配置，不使用 dev 包名和 dev 数据目录。
